<h2 style="text-align: center;"><span style="text-decoration: underline; color: #3366ff;">Vector Manipulation</span></h2>
Say we have a vector, vector A....
<ul>
	<li><span style="line-height: 14px;">If we multiply A by a positive constant, k, we are left with a new vector with the same direction as A, but with a new magnitude (kA)</span></li>
</ul>
<a href="http://compactcourse.com/wp-content/uploads/2013/06/Vector-kA.png"><img class="alignnone size-full wp-image-1466" style="margin: 5px auto; display: block; text-align: center;" alt="Vector kA" src="http://compactcourse.com/wp-content/uploads/2013/06/Vector-kA.png" width="500" /></a>
<ul>
	<li>If we multiply A by a negative constant, -k, we are left with a new vector with the opposite direction as A, and with a new magnitude (kA)</li>
</ul>
<p style="text-align: center;"><a href="http://compactcourse.com/wp-content/uploads/2013/06/Vector-kA1.png"><img class="alignnone size-full wp-image-1475" style="margin: 5px auto; display: block; text-align: center;" alt="Vector -kA" src="http://compactcourse.com/wp-content/uploads/2013/06/Vector-kA1.png" width="500" /></a></p>
Say we have another vector, vector B...
<ul>
	<li><span style="line-height: 14px;">If we add A to B, we are left with a new vector, vector A + B</span></li>
	<li>If A = A<sub>x </sub>+ A<sub>y </sub>and B = B<sub>x</sub> + B<sub>y</sub>, A + B = (A<sub>x</sub> + B<sub>x</sub>)i + (A<sub>y</sub> + B<sub>y</sub>)j</li>
</ul>
<a href="http://compactcourse.com/wp-content/uploads/2013/06/Vector-A+B.png"><img class="alignnone size-full wp-image-1486" style="margin: 5px auto; display: block; text-align: center;" alt="Vector A+B" src="http://compactcourse.com/wp-content/uploads/2013/06/Vector-A+B.png" width="500" /></a>
<ul>
	<li><span style="line-height: 22px;">If we subtract B from A, we are left with a new vector, vector A + (-B)</span><span style="line-height: 22px;"> </span></li>
	<li>If A = A<sub>x </sub>+ A<sub>y </sub>and B = B<sub>x</sub> + B<sub>y</sub>, A - B = (A<sub>x</sub> - B<sub>x</sub>)i + (A<sub>y</sub> - B<sub>y</sub>)j</li>
</ul>
